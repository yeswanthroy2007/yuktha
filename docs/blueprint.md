# **App Name**: Yuktha Health

## Core Features:

- Emergency Medical File: Collect and store user's emergency medical information (blood group, allergies, medications, emergency contact, etc.).
- Medicine & Pill Tracker: Smart reminders for tablets with time-based notifications; tracks dose confirmation (taken or skipped).
- AI Report Analysis: AI extracts key health parameters from uploaded lab reports. A tool compares values with past reports, highlighting changes and generating summaries with trend graphs.
- Emergency QR File: Generates a QR code for first responders to access a userâ€™s emergency medical file.
- Family Profiles: Parents can create and manage profiles for children or elderly family members, securely storing medical history, allergies, and immunizations.
- Prescription Management (User): Doctors upload prescriptions; users can pick up or request home delivery. Users will only be able to request pick up or delivery, can not add their own prescription.
- Prescription Management (Doctor): Doctors can upload and manage prescriptions for patients. This will be a dedicated Doctor UI for doctors only.

## Style Guidelines:

- Primary color: Deep teal (#008080), evokes trust and serenity, aligning with health and wellness.
- Background color: Very light teal (#E0F8F8), providing a calming backdrop.
- Accent color: Vibrant aquamarine (#7FFFD4), for interactive elements and highlights.
- Headline font: 'Poppins', a geometric sans-serif, conveys a contemporary and precise feel.
- Body font: 'PT Sans', a humanist sans-serif with a little warmth or personality; will be used for longer text
- Use clear, intuitive icons related to health, medicine, and family. Stick to simple, flat designs.
- Subtle transitions and animations to enhance user experience without being distracting.
